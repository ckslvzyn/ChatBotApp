# OpenChatBot

#### 介绍
微信机器人，机器人推送部分目前实现淘宝商品推送、定时任务、课表推送；另外支持企业微信机器人定时推送和ChatGPT的功能。微信机器人底层是openwechat——go结合C导出的动态库接口，C#+WPF做的接口交互和界面逻辑。
